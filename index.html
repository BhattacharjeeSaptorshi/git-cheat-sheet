Git Cheat Sheet

Compiled by - Saptorshi Bhattacharjee

Config
git config - used to set Git configuration values on a global or local project level.
Create
git init - creates a new Git repository
Workflow
git add - adds a change in the working directory to the staging area
git commit - saves the change from the staging area to local repository
git reset - moves both the HEAD and branch refs to the specified commit
Inspection and History
git status - displays the state of the working directory and the staging area
git diff - helps you see, compare, and understand changes in your project
git log - displays committed snapshots
git show - to view expanded details on Git objects such as blobs, trees, tags, and commits
File Operations
git clean - deleting untracked files in a repo's working directory
Branching
git checkout - the act of switching between different versions of a target entity
git revert - used for undoing changes to a repository's commit history
git stash - takes your uncommitted changes (both staged and unstaged), saves them away for later use, and then reverts them from your working copy
git branch - a pointer to a snapshot of your changes
git switch - allows you to switch your current HEAD branch
